# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

- プロジェクトに関するファイルやデータをまとめて保管しておく場所がリポジトリ。  
- 自分のPCに保存されているリポジトリ：ローカルリポジトリ  
- ネット上(github)に保存されているリポジトリ:リモートリポジトリ

## プッシュとマージの違いは何でしょうか？

- プッシュは、変更履歴(コミット)をリモートリポジトリに反映させる作業。  
- マージは、部品製造ライン(ブランチ)で作成した部品を、合体させる作業。

## コミットとプッシュの違い

- コミットは、リモートリポジトリでの変更履歴を残す作業。
- プッシュは、その変更履歴をリモートリポジトリに反映させる作業。


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

- 一目で変更内容がわかるように書く。

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

- masterにマージしてからプッシュするか、プッシュしてからマージするか。


## コンフリクトを起こしてしまった場合、どう対処すべきですか？

- コンフリクトを起こした他人のコードを把握したり、PMと相談をし、何を取り込むべきなのかをしっかり判断する。
